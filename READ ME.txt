1. Copy this entire folder to the htdocs in xampp folder
2. Start the apache and mysql server in xampp control panel
3. Go to http://localhost/drive
4. Go to http://localhost/phpmyadmin
5. Create a new database and name it 'drive' (without the quotes)
6. Click on import and import the drive.sql file in the drive folder
7. Create an account in drive and play around with it
